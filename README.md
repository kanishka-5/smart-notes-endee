**🧠 NeuroNotes AI — Smart Thinking. Faster Answers. ⚡**
**🚀 Your Personal Knowledge Engine**

A simple AI-powered note search application with login authentication and smart search functionality. Built using Streamlit and deployed on Streamlit Cloud.

---

## 🚀 Features

- 🔐 User Login System
- 🏠 Home Page Navigation
- 🔍 Smart Search (Top 3 results)
- 📂 Text-based data search
- ⚡ Fast and lightweight
- 🌐 Deployed online using Streamlit Cloud

---

## 🧠 How It Works

1. User logs in using credentials
2. App loads notes from `data/data.txt`
3. User enters a query
4. App matches keywords and returns top results
